# 🧪 WhatShouldIDo

> **An AI-assisted playground for implementing various web features without constraints.**
> **AI를 적극 활용하여, 형식에 구애받지 않고 다양한 기능을 구현하고 모아두는 실험적인 웹 애플리케이션입니다.**

## 🚀 Project Concept
이 프로젝트는 단순히 하나의 서비스를 위한 것이 아닙니다. 개발자가 흥미를 느끼는 다양한 기능(Feature)들을 **AI(LLM)와 협업**하여 빠르게 구현하고 테스트하는 **'All-in-One' 플랫폼**을 지향합니다.

현재 첫 번째 메인 기능으로 **사용자 취향 기반 영화 추천 서비스**가 구현되어 있으며, 앞으로 다양한 유틸리티와 실험적인 기능들이 추가될 예정입니다.

## 🌟 Current Features
### 1. Movie Recommendation (영화 추천) 🎬
- **Description**: 사용자가 선택한 영화(5개 이상)의 취향 벡터를 분석하여 TMDB 데이터를 기반으로 최적의 영화를 추천합니다.
- **Tech**: Vector Similarity, TMDB API, Content-based Filtering
